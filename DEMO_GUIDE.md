# TipMind — Judge Demo Script

**Total demo time: ~90 seconds**

> Run `make demo` before the presentation to seed data and launch both servers.

---

## Pre-Demo Checklist

- [ ] `make demo` is running (FastAPI :8000 + Next.js :3000)
- [ ] Browser open at http://localhost:3000
- [ ] Dashboard shows wallet balance and 4 metric cards with data
- [ ] Two swarm cards visible (one at 80% progress)
- [ ] Live activity feed shows recent agent decisions

---

## The Script

### Step 1 — Agent Settings Panel (~10 s)

**Say:** *"TipMind is an autonomous AI agent that tips crypto to video creators based on real-time signals. Here you can configure it — max tip per video, preferred token, and which triggers are active."*

**Do:**
- Point to the **Agent ON/OFF toggle** (pulsing green dot = active)
- Point to the **Max per video slider** ($0–$10 range)
- Point to the **Token selector** (USDT / XAUT / BTC)
- Point to the **Trigger toggles** (Watch Time ✓, Chat Emotion ✓, Milestones ✓, Swarms ✓)

---

### Step 2 — Simulate Watch (~10 s)

**Say:** *"When a user watches 80% of a video, TipMind's Watch Agent kicks in. Claude sizes the tip based on engagement depth and fires it automatically."*

**Do:**
- Click **[Simulate Watch]** in the Demo Control Bar
- Watch the **Live Activity Feed** — a blue entry slides in with reasoning like *"Watch completion at 80% signals high quality"*
- Transaction table updates with a new WATCH_TIME tip

---

### Step 3 — Inject Hype (~10 s)

**Say:** *"During live streams, TipMind monitors chat sentiment in real time. When the crowd goes wild, the Emotion Agent detects the spike and rewards the creator."*

**Do:**
- Click **[Inject Hype]** in the Demo Control Bar
- Watch the **Live Activity Feed** — a yellow entry appears: *"Chat hype velocity: 18 positive messages in 30s. Emotion threshold crossed."*
- Metrics update: Tips Sent counter increments

---

### Step 4 — Fire Milestone (~15 s)

**Say:** *"Creators hit milestones — winning debates, crossing 100K views, subscriber records. TipMind celebrates these moments automatically with a premium tip and a custom message generated by Claude."*

**Do:**
- Click **[Fire Milestone]** in the Demo Control Bar
- Watch the **Live Activity Feed** — a purple milestone entry appears
- Point out the **Largest Single Tip** metric updating (milestone tips are bigger)
- Point out the **Transaction Table** — shows the milestone tip with trigger type `MILESTONE`

---

### Step 5 — Release Swarm 🚀 (~20 s) — MAIN MOMENT

**Say:** *"This is the flagship feature. A fan swarm is a collective tip pool — multiple fans pledge to tip simultaneously when a trigger fires. Watch what happens when the DEBATE_WIN event lands."*

**Do:**
- Point to the **Swarm Card** showing Alex Rivera's pool at 80% — *"Tip $50 if Alex wins the debate tonight"*
- Click **[Release Swarm]** in the Demo Control Bar
- Watch the **Swarm Card explode** with a Framer Motion animation (scale + glow)
- Watch the **Live Activity Feed** — a red entry: *"SWARM RELEASED: 20 fans tipped $100 simultaneously"*
- Watch the metrics: Total Tipped, Tips Sent, and Creators Supported all update

**Say:** *"Every participant's tip fired in parallel using asyncio.gather() — one atomic burst on-chain."*

---

### Step 6 — Transaction Table (~10 s)

**Say:** *"Every tip is recorded with a full transaction hash. In production these are real on-chain transactions via Tether's WDK."*

**Do:**
- Scroll down to the **Transaction Table**
- Point out the columns: time, creator, amount, token, trigger type, tx hash
- Click a tx hash to show it's formatted as a blockchain explorer link

---

### Step 7 — Wallet Balance (~10 s)

**Say:** *"The agent tracks wallet balance in real time. After the swarm release, you can see the balance has decreased — real USDT moved to the creator's wallet via WDK."*

**Do:**
- Point to the **wallet balance** in the header (USDT balance)
- Point to the **Wallet Address** (truncated 0x...)

**Say:** *"TipMind is fully autonomous — it watches, decides, and tips without any manual action from the fan."*

---

## Key Talking Points

| What | Why it matters |
|---|---|
| **Claude decides tip amounts** | Not rule-based — the AI sizes each tip contextually |
| **asyncio.gather() swarms** | All fan tips fire simultaneously, not sequentially |
| **WDK integration** | Real USDT on Ethereum L2 — not toy tokens |
| **WebSocket live feed** | Judges see decisions happening in real time |
| **4 independent agents** | Each specializes in one signal type; Orchestrator coordinates |

---

## If Something Goes Wrong

| Problem | Fix |
|---|---|
| Dashboard shows no data | Run `make seed` then refresh |
| WebSocket feed not updating | Check backend is running on :8000 |
| Demo buttons not responding | Check browser console for 404 — verify Next.js rewrites in next.config.ts |
| Swarm animation doesn't fire | Refresh page; animation requires the SWARM_RELEASED WebSocket event |
